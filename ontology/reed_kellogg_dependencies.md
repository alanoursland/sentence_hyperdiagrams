# Reed-Kellogg Label Dependencies

This note sketches dependencies among labels in `reed_kellogg.yaml`.
It is not a grammar. It is a compact planning view for ordering annotation
passes and deciding which labels can be derived from earlier labels.

## Edge convention

`LABEL <- A, B` means `LABEL` depends on, contains, or refines `A` and `B`.

Two kinds of dependency are mixed deliberately:

- `is-a`: a label is a narrower form of another label.
- `builds-from`: a label is a constituent built from other labels.

## Is the graph a DAG?

Not as a raw grammar graph.

Reed-Kellogg grammar is recursive:

- A `NOUN` projection can take an `ADJECTIVE_MODIFIER`.
- An `ADJECTIVE_MODIFIER` can be an `ADJECTIVE_PHRASE`.
- An `ADJECTIVE_PHRASE` is often a `PREPOSITIONAL_PHRASE`.
- A `PREPOSITIONAL_PHRASE` contains a `PREPOSITIONAL_PRINCIPAL`.
- A `PREPOSITIONAL_PRINCIPAL` is normally a nominal, so it returns to `NOUN`.

That gives a real construction cycle:

```text
NOUN -> ADJECTIVE_MODIFIER -> ADJECTIVE_PHRASE
     -> PREPOSITIONAL_PHRASE -> PREPOSITIONAL_PRINCIPAL -> NOUN
```

Clauses also recurse: dependent clauses can function as noun, adjective, or
adverb constituents inside larger clauses.

If recursive modifier, phrase, compound, and clause attachment are treated as
iteration inside a layer, the quotient graph is a DAG. That layered DAG is the
useful pass structure.

## Practical pass order

```text
01_pos
  token POS and punctuation labels

02_lexical_refinement
  POS subtypes, possessives, verb types, articles, pronoun types, verbals

03_projections
  nominal projection, verb phrase, simple subject, simple predicate,
  simple complements, simple prepositional principal

04_modifiers_and_phrases
  modifier attachment, prepositional phrases, infinitive and participle
  phrases, adjective and adverb phrase use

05_compounds
  compound subject, predicate, complement, and prepositional principal

06_clauses
  independent and dependent clauses, noun/adjective/adverb clause use

07_sentences
  sentence type, sentence complexity, independent elements
```

The current `reed_kellogg_02_primitives.yaml` covers part of layer 3: nominal
projection, simple and compound subjects, simple and compound object
complements, and simple and compound prepositional principals.
