# Roget's Thesaurus

**Original author:** Peter Mark Roget (1852)

## What it is

A thesaurus organized not alphabetically but by **conceptual categories**.
Words are grouped under ~1,000 headwords arranged in a six-class hierarchy:

1. Abstract Relations
2. Space
3. Matter
4. Intellect
5. Volition
6. Affections

Each class subdivides into sections, then headwords, then synonyms/related
words grouped by part of speech.

## Relevance to CLib / hyperdiagrams

CLib used Roget's headwords as one source for **naming components** — choosing
intuitive, natural English names for knowledge components. For the
hyperdiagram project:

- The headword hierarchy provides a conceptual organization scheme for label
  names
- The six top-level classes offer a coarse ontological structure
