# Longman Dictionary of Contemporary English (LDOCE)

**Publisher:** Pearson / Longman

## What it is

An English learner's dictionary notable for its **defining vocabulary** — a
restricted set of ~2,000 common English words used to write all definitions
in the dictionary. This constraint ensures that every word in the language
can be explained using only these ~2,000 words.

## Relevance to CLib / hyperdiagrams

CLib used the Longman defining vocabulary as a source for **naming
components** — if a concept can be expressed using one of these ~2,000 basic
words, it's likely to be intuitive and widely understood. For the
hyperdiagram project:

- The defining vocabulary is a candidate source for label names — a small,
  closed set of words sufficient to describe any concept
- The principle (small vocabulary, broad coverage) parallels the hyperdiagram
  goal of a compact label ontology with broad linguistic coverage
