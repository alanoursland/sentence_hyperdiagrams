# Reed-Kellogg Sentence Diagrams

**Authors:** Alonzo Reed, Brainerd Kellogg
**Original publications:**
- *Higher Lessons in English* (1877)
- *Graded Lessons in English* (1896)

## What it is

The Reed-Kellogg system is a visual method for diagramming English sentence
structure, developed in the late 19th century and widely taught in American
schools through the mid-20th century. It represents grammatical relationships
through spatial positioning and line conventions on a two-dimensional diagram.

### Core sentence elements

- **Subject** — names that of which something is thought
- **Predicate** — tells what is thought (the verb)
- **Object complement** — completes predicate; names what receives the act
- **Attribute complement** — completes predicate; asserts quality/class of
  subject (predicate nominative or predicate adjective)

### Modifier types

- **Adjective** — modifies noun/pronoun (quality, pointing-out, number)
- **Adverb** — modifies verb/adjective/adverb (manner, time, place, degree)
- **Prepositional phrase** — preposition + noun + modifiers, functioning as
  adjective phrase or adverb phrase

### Compound structures

- **Compound subject** — multiple subjects sharing one predicate
- **Compound predicate** — multiple predicates sharing one subject
- **Compound complement** — multiple objects or attributes

### Phrase types

- **Prepositional phrase** — preposition + object + modifiers
- **Adjective phrase** — phrase functioning as adjective
- **Adverb phrase** — phrase functioning as adverb
- **Verb phrase** — auxiliary + main verb (is learning, could have been)

### Eight parts of speech

1. Noun
2. Pronoun
3. Verb
4. Adjective
5. Adverb
6. Preposition
7. Conjunction
8. Interjection

### Diagram conventions

| Element | Visual representation |
|---------|----------------------|
| Subject / Predicate | Horizontal baseline, divided by vertical bar |
| Object complement | After verb, separated by short vertical line |
| Attribute complement | After linking verb, separated by backslash |
| Word modifiers | Slanting lines below the word modified |
| Prepositional phrases | Slanting line (preposition) + horizontal (object) |
| Conjunctions | Dotted lines connecting compound elements |
| Subordinate clauses | On separate baselines, connected to main diagram |

## Relevance

Reed-Kellogg provides the **first batch of Parts of Thought labels**. Its
structural elements map directly to label names:

- Subject, Predicate, Object, Attribute → linking labels
- Noun, Verb, Adjective, Adverb, etc. → leaf labels (POS)
- Prepositional phrase, Adjective phrase → phrase-level labels
- Compound structures → coordination labels

The system was designed for exactly the kind of text in the early reader
corpus (19th-century primer English). Starting with Reed-Kellogg labels
ensures the ontology handles the dataset's core structures before expanding
into deeper semantics.
