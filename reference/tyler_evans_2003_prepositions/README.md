# The Semantics of English Prepositions: Spatial Scenes, Embodied Meaning, and Cognition

**Authors:** Andrea Tyler, Vyvyan Evans
**Published:** Cambridge University Press, 2003
**ISBN:** 978-0-521-04633-0

## What it is

The gold standard for cognitive-linguistic analysis of English prepositions.
Explains how spatial scenes (physical arrangements of entities) map
systematically to abstract meanings through embodied cognition.

### Key ideas

- Each preposition has a **proto-scene** — a core spatial configuration
  (e.g., "over" = an entity above and across a landmark)
- Abstract senses derive from the proto-scene through systematic
  **conceptual metaphor** (e.g., "she got *over* the cold" maps recovery
  onto the trajectory of crossing above an obstacle)
- Detailed analyses of major prepositions: over, above, below, under,
  in, out, through, to, for, at, up, down, etc.
- Each preposition's sense network is mapped as a radial category

## Relevance

Prepositions are the primary surface markers of semantic relations in the
hyperdiagram. This book provides the logic for mapping from syntactic
preposition labels to semantic-level labels:

- **Place-to-Place slots** (is-above, is-inside, is-near, etc.) derive
  from spatial proto-scenes described here
- **Event-to-Entity slots** (destination, origin, path, toward, away-from)
  are the abstract extensions of spatial prepositions
- Understanding which abstract sense a preposition triggers determines
  which semantic label to assign
