# Semi-Automatic Recognition of Semantic Relationships in English Technical Texts

**Author:** Ken Barker
**Published:** PhD Dissertation, University of Ottawa, Department of Computer Science, 1998
**Supervisor:** Stan Szpakowicz

## Summary

Barker's dissertation provides a thorough analysis of case roles and semantic
relations — the work that directly underpins the Component Library's relation
dictionary. It develops a semi-automatic system for identifying semantic
relationships in noun phrases without requiring pre-coded noun or adjective
semantics.

### Key contributions

- A comprehensive taxonomy of semantic relations between nouns and their
  modifiers, organized into systematic categories
- A semi-automatic recognition system that correctly identifies 60-70% of
  relationships automatically
- Analysis grounded in case role theory (Fillmore) extended to cover
  entity-entity, event-entity, and other relation types

### Published version

The ACL'98 paper "Semi-Automatic Recognition of Noun Modifier Relationships"
(Barker & Szpakowicz, 1998) presents the core ideas from this dissertation.

## Relevance

This is the **foundation** for the CLib semantic relations dictionary, which
in turn is the primary source for hyperdiagram label names at the semantic
level. The relation taxonomy developed here defines the categories used across
all the Porter group's later work.
